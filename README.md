# GCP-DevOps

This repository contains notes, experiments, and small projects related to Google Cloud Platform, DevOps, CI/CD, Terraform, GitHub Actions, and learning-to-code exercises.

## Repository Structure

- [devlogs/](devlogs/) - Development logs and notes from my DevOps journey
- [gcp-projects/](gcp-projects/) - Google Cloud projects and related implementations
- [github-actions/](github-actions/) - GitHub Actions workflows and examples
- [learning-to-code/](learning-to-code/) - Learning materials for Bash, PowerShell, and Python
- [terraform/](terraform/) - Terraform projects and infrastructure-as-code examples
- [test-files/](test-files/) - Sample files and test projects
- [LICENSE](LICENSE) - Repository license
- [GCP-DevOps.code-workspace](GCP-DevOps.code-workspace) - VS Code workspace configuration

## Purpose

This repo is organized as a practical learning space for exploring cloud infrastructure, automation, and software development workflows.
